The easiest carousel, Quickousel.

Shit images into a div, get a responsive carousel.

      • Javascript (optional)

      // Copy && Paste
      Quickousel.init({
      "transitionStyle":"fadeIn",
      "transitionSpeed": 8000,
      "arrows": false,
      "button": "dottedsquare"
      });






      For all options, view the github page.

      HTML

      <!-- Add the id of "quickousel" to any element -->
      <div id="quickousel">
      <!-- add the images of your choice, thats it. -->
         <img src="image1.jpg"/>
         <img src="image2.jpg"/>
         <img src="image3.jpg"/>
         <img src="image4.jpg"/>
      </div>
